## 1. Тестування коду з ЛР № 5. Завдання 1 

**Мета:** Перевірити роботу алгоритму кореляції рядів $A$ та $B$ при різних значеннях $N$.

| № Тесту | Опис тесту | Вхідні дані (Input) | Очікуваний результат (Output) |
| :---: | :--- | :--- | :--- |
| **1** | **Позитивний взаємозв'язок**<br>(Дані змінюються синхронно) | `N: 3`<br>`A: 10, 20, 30`<br>`B: 2, 4, 6` | `mean(A)= 20.000000`<br>`mean(B)= 4.000000`<br>`K1= 1, K2= 1, K= 0.666667`<br>Висновок: `Relations between series A and series B is real` |
| **2** | **Відсутність взаємозв'язку**<br>(Дані змінюються хаотично або різноспрямовано) | `N: 3`<br>`A: 10, 20, 30`<br>`B: 6, 4, 2` | `mean(A)= 20.000000`<br>`mean(B)= 4.000000`<br>`K1= 0, K2= 0, K= 0.000000`<br>Висновок: `Relations between series A and series B is not real` |
| **3** | **Граничний випадок**<br>(Введення нуля) | `N: 0` | Вивід: `Enter N first`<br>*(або завершення без помилок, залежно від реалізації перевірки)* |

## 3. Тестування коду з ЛР № 6

**Мета:** Перевірити регулярний вираз та логіку пошуку слів, що починаються і закінчуються однією літерою.

| № Тесту | Опис тесту | Вхідні дані (Input) | Очікуваний результат (Output) |
| :---: | :--- | :--- | :--- |
| **1** | **Стандартний текст**<br>(Є слова, що підходять) | Text: `"Anna and Bob went to the gig at noon"` | **Знайдені слова:**<br>`anna`<br>`bob`<br>`gig`<br>`noon` |
| **2** | **Текст без збігів** | Text: `"Hello world this is a test"` | `These words were not found` |
| **3** | **Різний регістр та розділові знаки** | Text: `"Level! Radar? Stats."` | **Знайдені слова:**<br>`level`<br>`radar`<br>`stats` |
| **4** | **Однолітерні слова**<br>(Мають ігноруватися за умовою `len > 1`) | Text: `"a I o u"` | `These words were not found` |

## 4. Тестування коду з ЛР № 7 

**Мета:** Перевірити роботу меню, додавання, пошук та видалення елементів словника.

| № Тесту | Опис тесту | Вхідні дані (Дії в меню) | Очікуваний результат |
| :---: | :--- | :--- | :--- |
| **1** | **Пошук за назвою**<br>(Існуючий елемент) | `Меню: 1`<br>`Name: Malware` | `Found: Malicious software.` |
| **2** | **Пошук за ключовим словом**<br>(В описі) | `Меню: 2`<br>`Keyword: software` | **Found threats:**<br>`- Ransomware`<br>`- Malware` |
| **3** | **Додавання нового елементу** | `Меню: 3`<br>`Name: Spyware`<br>`Description: Spies on user.` | `New threat 'Spyware' successfully added.` |
| **4** | **Видалення елементу** | `Меню: 4`<br>`Name: DDoS` | `Threat 'DDoS' was successfully deleted.` |
| **5** | **Перевірка списку**<br>(Після видалення та додавання) | `Меню: 5` | Відображається список, де є `Spyware`, але немає `DDoS`. |
